# Аналіз ефективності алгоритмів пошуку підрядка

Проведено порівняння трьох класичних алгоритмів пошуку підрядка: **Боєра-Мура**, **Кнута-Морріса-Пратта (КМП)** та **Рабіна-Карпа**. Тестування здійснювалось на базі двох великих текстових файлів (Стаття 1 та Стаття 2) за допомогою вбудованого модуля `timeit` (кількість повторень кожного тесту  - 50).

Для кожного тексту було протестовано два сценарії:
1. Пошук підрядка, який **існує** в тексті (ближче до кінця файлу).
2. Пошук **вигаданого** підрядка (найгірший випадок, коли алгоритм повинен повністю просканувати весь об'єм тексту).

```python
tests = {
    "Стаття 1": {
        "text": text1,
        "existing": "найбільш популярних алгоритмів",
        "fake": "hash map",
    },
    "Стаття 2": {
        "text": text2,
        "existing": "linked list",
        "fake": "жадібні алгоритми",
    },
}
```

---

### Результати замірів часу

| Текст | Тип | Boyer-Moore(s) | KMP (s) | Rabin-Karp (s) |
| :--- | :--- | :--- | :--- | :--- |
| **Стаття 1** | existing | **0.00241** | 0.01530 | 0.03227 |
| **Стаття 1** | fake | **0.00708** | 0.01550 | 0.03149 |
| **Стаття 2** | existing | **0.00573** | 0.02173 | 0.05319 |
| **Стаття 2** | fake | **0.01728** | 0.06865 | 0.15224 |


### Аналіз результатів

Для обох статей найшвидшим виявився алгоритм Боєра-Мура.
Алгоритм Боєра-Мура продемонстрував перевагу над конкурентами. При пошуку існуючого підрядка він виявився швидшим за КМП приблизно в 4.5 рази, а за Рабіна-Карпа - майже в 12 разів. На вигаданому підрядку швидкість Боєра-Мура трохи зросла, через відсутність збігів, проте алгоритм обробив все ще швидше за КМП та Рабіна-Карпа.

Для стандартного пошуку підрядка в текстових документах та статтях алгоритм **Боєра-Мура** є найкращим та найефективнішим вибором.
